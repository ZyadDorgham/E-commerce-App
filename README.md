# 🛒 E-Commerce Project

This is a modern E-commerce web application built with React and Vite. The project allows users to browse products, view product information, and manage their shopping cart through a clean and responsive user interface.

## 🚀 Features

* Browse a collection of products
* Add products to the shopping cart
* Remove products from the cart
* Update product quantities
* Dynamic cart management
* Responsive design for all devices
* Clean and user-friendly UI

## 🛠️ Tech Stack

* React.js
* Vite
* JavaScript (ES6+)
* CSS3

## ⚙️ How It Works

The application displays a list of products that users can browse and add to their cart. Cart data is managed dynamically, allowing users to update quantities, remove items, and view their selected products in real time.
